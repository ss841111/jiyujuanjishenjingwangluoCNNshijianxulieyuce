## 评价指标说明

- **R2**：决定系数，衡量模型对数据的拟合程度。
- **MAE**：平均绝对误差，衡量预测值与实际值之间的平均绝对差异。
- **MSE**：均方误差，衡量预测值与实际值之间的平方差的平均值。
- **RMSE**：均方根误差，MSE的平方根，衡量预测值与实际值之间的标准差。
- **MAPE**：平均绝对百分比误差，衡量预测值与实际值之间的平均百分比差异。
